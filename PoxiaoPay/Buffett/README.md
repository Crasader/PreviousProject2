Buffett
=======

Buffett是一个Android平台上的支付中间层。
用于隔离游戏开发的支付逻辑与各种支付插件。

VERSION = "8"
*DO1 配合TAT的更新进行接口的调整
*DO2 移除从服务端获取插件的接口,单独为一个工程
